Report serious misconduct by a teacher in England: step-by-step

The Teaching Regulation Agency (TRA) is responsible for investigating allegations of serious misconduct against teachers in schools in England. If necessary, it can stop individuals from teaching.

This is how to report serious misconduct to TRA. There’s a different process if you’re in Wales, Scotland or Northern Ireland.

There’s separate information about what to expect if you’re a teacher who has been accused of serious misconduct.

    {% set stepNumber = 1 %} {% set stepTitle = 'Make an informal complaint' %} {% set stepHtml %}

    If you’re a member of the public, complain to ​​the school, school governors or your local council before making a formal complaint to TRA.

    If you’re an employer, follow your organisation’s internal disciplinary procedures first.

    {% endset %} {% include "report/_step.html" %} {% set stepNumber = 2 %} {% set stepTitle = 'Check if TRA will deal with the complaint' %} {% set stepHtml %}

    If you’re not satisfied with the response to your informal complaint, or you think the misconduct is serious enough that the person should no longer teach, check if the complaint can be dealt with by TRA.

    Serious misconduct is unacceptable behaviour that:

    • falls significantly short of the standards expected of a teacher
    • brings the teaching profession into disrepute
    • affects the way someone performs their teaching role

    If the person is convicted of a relevant offence at any time, this is also serious misconduct.

    See who you can report to TRA

    {% endset %} {% include "report/_step.html" %} {% set stepNumber = 3 %} {% set stepTitle = 'Report the teacher to TRA' %} {% set stepHtml %}

    You can use the Report a teacher for serious misconduct service or complete and return a downloadable form. There’s a form for employers and a form for members of the public.

    You’ll need to provide evidence.

    {% endset %} {% include "report/_step.html" %} {% set stepNumber = 4 %} {% set stepTitle = 'Wait to hear back from TRA' %} {% set stepHtml %}

    Once TRA have reviewed the report, they’ll let you know what happens next. TRA could tell you they:

    • need more information to decide whether to investigate the case
    • will not investigate the case because it’s outside their remit
    • will not investigate the case because it does not have the potential to result in a prohibition order
    • will launch a formal investigation

    TRA might also consider imposing an interim prohibition order. This prevents the individual from teaching until the case has concluded.

    {% endset %} {% include "report/_step.html" %} {% set stepNumber = 5 %} {% set stepTitle = 'If TRA decide to investigate, they’ll send a copy of the allegations to the teacher' %} {% set stepHtml %}

    TRA will share the allegations with the teacher and ask them to respond. They might also appoint a legal firm to help with the investigation.

    If the teacher admits to the allegations, they can ask to have their case heard at a meeting instead of a hearing.

    {% endset %} {% include "report/_step.html" %} {% set stepNumber = 6 %} {% set stepTitle = 'TRA decide whether there should be a formal meeting or hearing' %} {% set stepHtml %}

    Once the deadline for a response from the teacher has passed, a TRA decision maker will consider the case. They’ll take into account all the evidence, including information from previous employers or the police.

    If the decision maker concludes that the misconduct is not serious enough, or that there’s no case to answer, they’ll discontinue the case.

    If TRA believe a prohibition order could be made, they’ll arrange a professional conduct panel meeting or a hearing. The panel is made up of 3 independent people who are not employed by TRA or the Department for Education (DfE).

    {% endset %} {% include "report/_step.html" %} {% set stepNumber = 7 %} {% set stepTitle = 'A professional conduct panel meeting or hearing takes place' %} {% set stepHtml %}

    If there’s a meeting

    If there’s a meeting, the panel will meet in private to consider the case, assisted by a legal adviser. There are no witnesses and the presenting officer and teacher do not attend.

    A public notice of a meeting is published on GOV.UK website one week in advance.

    If there’s a hearing

    Hearings are attended by:

    • panel members
    • a presenting officer, who is a lawyer
    • a legal adviser, who is a lawyer,
      • independent of TRA
    • the person accused of serious misconduct
    • a representative for the accused
    • witnesses
    • TRA staff

    Members of the public, including the press, can attend both virtual and in-person hearings as observers.

    A public notice of a hearing is published on GOV.UK website one week in advance.

    {% endset %} {% include "report/_step.html" %} {% set stepNumber = 8 %} {% set stepTitle = 'TRA confirm what action they’ll take' %} {% set stepHtml %}

    If the professional conduct panel makes a finding of serious misconduct, they’ll make a recommendation about whether a teacher should be prohibited. The final decision is then made by a senior TRA official.

    The person who submitted the original report will normally be informed in writing within 2 working days of the decision being made.

    If there’s a finding of serious misconduct, a document summarising the outcome will be published online within 2 weeks.

    Teachers have 28 days to appeal prohibition orders made against them.
